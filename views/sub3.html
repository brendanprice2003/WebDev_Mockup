Technologies Utilized During Development

Outlining Technologies

Below is a list of technologies that may be used during web development.
  • HTML, CSS
  • JavaScript
  • PHP
  • MySQL
  • Scss/Sass

Dissimilarities of Specific Techniques

In the previous section, I had outlined the dissimilarities between online website creation tools and custom building websites, in this section I will explain the differences between the specific technlogoie and techniques that are used in both instances of website creation.

Languages such as HTML, CSS are not considered to be languages, they are instead considered to be a markdown language. This means that they solely based on providing a framework for a runtime to make use of. In this case that runtime would be JavaScript (in most cases); JavaScript provides websites with most, if not all, of the user interaction, user experience, and behind the scenes functions. The key difference between HTML/CSS and JavaScript is that HTML/CSS is a markdown language which just provides a framework for website objects, whereas JavaScript is a programming language that is capapable of logical calculations and interfaces with the browser.

PHP is another programming language, like JavaScript, it can execute logical calculations and interface with the browser excepit it is capable of doing server side processing too. PHP accepts the use of HTML/CSS and JavaScript and many other common web development solutions all within the syntax. A web server that runs PHP will render the website on the server, passing any required data to the website, then sending it to the client. This is vastly different to the conventional HTML/CSS and JavaScript setup because in that scenario the client is just requesting the specified files and "that's that". However, PHP will render the website BEFORE sending to the client which allows it to append data, adding another platform at which the client is able to interface with.

MySQL is a server language, but more modern solutions allow for the use of MySql on the client (in the browser). MySQL is a language that revolves around interfacing with a specified database to request and received data based on a defined set of requirements. MySQL is specialised and excels in this area compared to other languages when interfacing with databases. Other languages may be able to interface with databases, yes, but MySQL provides a solution to the lack of support and/or quality in those languages. MySQL is only used for interfacing with databases and therefore is not able to do things like JavaScript, such as changing the innerHTML property of a <p> tag.

Scss or Sass is a CSS pre-processor; They take the fundamentals of CSS and is able to add more working parts to it. Scss or Sass is able to append enough functionalities that CSS can actually be considered a programming language in some cases. Using functions such as if statements and booleans to characterize properties for object styling. CSS is not currently able to make explicit comparisons using if statements or even use booleans to express values on properties. That is the key difference between CSS and Scss/Sass.
