(My) Preference of Website Technologies

Justification of Website Creation Technologies

Every developer has their strengths when it comes to programming and web development, that is why I have curated a list of my preferences for tools and techniques that are commonly utilized within web development and programming alike. Along with each prefered tool and/or technique I will provide a jusification for this preference.

Tools and Software

Below is a list of programming languages (or markdown languages) that I prefer to utilize. Note: React.js is a JavaScript framework that allows for a dynamic development environment.
  • HTML, CSS
  • JavaScript
  • React.js
  • Scss/Sass
Justification

HTML,CSS and JavaScript all go hand-in-hand and is the simplest form of web development tool (for myself), consists of all required functionalities for adequate user experiences, whilst not overcomplicating concepts. Finding a suitable platform build your website on is key and this is why I prefer to use HTML,CSS and JavaScript for web development.

React.js and Scss/Sass take what is already good about CSS and JavaScript and add even more functionality to it. This extra functionality may be key for the web application to achieve a certain degree of user experience etc.
